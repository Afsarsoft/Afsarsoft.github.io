HTML (Hyper Text Markup Language) describes the structure of a Web page.
HTML tags tell the browser how to display the content such as "heading", "paragraph", "table", ... on the web page.
The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and display them. HTML tags are surrounded by angle brackets.
In summary, HTML defines the content of web pages.
